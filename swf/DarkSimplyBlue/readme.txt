-=[ INSTALLATION ]=-

  -=- Unzip DarkSimplyBlue.zip on your themes folder.
  -=- If you want to disable the topbar edit topbar.php.
  -=- If your site is on a subdomain (http://www.yoursite.domain.com) or in a folder (http://www.domain.com/yoursite) You'll need to edit style.css under themes/DarkSimplyBlue/style/ folder.
   Change all lines:
    background-image : url(/themes/DarkSimplyBlue/images/image.gif)

   to:
    background-image : url(http://[URL]/themes/DarkSimplyBlue/images/image.gif)

  where [URL] is the full address to you site.
