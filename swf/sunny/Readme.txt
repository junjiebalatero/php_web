Name of Addon/Theme/Mod: 
=========================
Sunny v1.0 - Theme for PHP Nuke version 5.0 beta 4 and above
Best working in IE 5.0 and above



Files included:
=================

/Readme.txt - Installation and Copyright Informations
themes/sunny/theme.php - Main theme file
themes/sunny/images - Images for this theme
themes/sunny/style - Style Sheet for this theme


Installation:
==============

Enjoy, the interest in this theme is the option let you include banners.php replacing the searchs boxes in the top.
