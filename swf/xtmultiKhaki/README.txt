xtmultiKhaki myPHPNuke Theme
Created by Xend (http://www.media.xendtech.com)


To Install: simply upload the xtmultiKhaki folder to your nuke/themes/ directory.


No php files were hurt during the creation of this theme.
No hardcoded colors are used in this theme.

All colors and font calls are CSS based.  If you would like to change the style of your site, edit the styles/style.css file.
You can edit "topheader.php" if you like to change the link's.
