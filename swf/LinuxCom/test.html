    New PHP Updates
    Posted on Sunday, November 12 @ 12:41:32 CST [ Edit | Delete ] I have posted updates to my PHP 4.0.3pl1 RPMs to ftp://ftp.e-smith.org/pub/e-smith/contrib/DanBrown/RPMS/i386/. The biggest difference is that the php base module now includes support for freetype. I have also built a php-gd module, which will add support for gd. These changes were made so that the Typo3 Content Management System could be used. The Webmail HOWTO, below, has been edited to reflect this change.
